Help with Flash | Click here

Since flash was deprecated in 2021, here are some ways you can activate it:

For Those who use old versions of Chrome or Edge:

  1. Click the pop-up that says "Flash is blocked on this site"
  2. Go on settings and activate flash
  3. Reload the page (Game might load anyway so you can ignore this step)

Please be reminded that every time you have to boot up chrome, you have to do this again. And we really do not recommend using older browser or even flash in general.

For those who use new versions of browsers:

  1. Go to the chrome web store
  2. Search for Ruffle
  3. Download it
  4. Test flash using ultrasounds.com

Note that Ruffle has limited support for Actionscript 3.

Cherish these games like how you used to.
